# Inabat Cherry Grove Portfolio

Готовый статический сайт-портфолио для конкурса CAP Education «Обо мне».

## Структура

- `index.html` — разметка сайта
- `style.css` — дизайн, адаптация и анимации
- `script.js` — интерактивность, винилы, плеер, лабиринт
- `assets/images` — локальные визуальные элементы
- `assets/audio` — место для аудио

## Музыка

Кнопка Play сначала ищет файл:

```text
assets/audio/see-you-again.mp3
```

Если файла нет, сайт не падает и включает короткий безопасный 8-bit preview. Чтобы звучала настоящая песня, нужно положить свой легально полученный аудиофайл в эту папку с указанным именем.
